#  MUIP GUI — Панель управления для GIO сервера

 Веб-интерфейс для управления игровым сервером через MUIP API с поддержкой GM-команд, выдачи предметов, управления персонажами и многим другим.

##  Основные возможности

- **Выдача любых предметов** — оружие, артефакты, материалы, персонажи
- **Спавн монстров** с указанием количества и уровня
- **Управление квестами** — добавление, принятие, завершение
- **Телепорт** по координатам, в сцены и подземелья
- **Отправка почты** с вложениями (поддержка item_list)
- **Быстрые GM-команды** — режим бога, бесконечная стамина, энергия без КД
- **Справочник ID** — интерактивный поиск по всем предметам и сущностям
- **Автоматическая подпись запросов** (SHA256 + сортировка параметров)
- **Поддержка всех MUIP функций** (100+ обработчиков)
- **Система уведомлений** и **консоль** с логами ответов сервера
- **Сохранение пользовательских команд** в data.json
- **Компактный режим сайдбара** для удобной работы на малых экранах

##  Технологии

- **Backend**: Python 3 + Flask
- **Frontend**: HTML5, CSS3, Vanilla JS
- **Шрифты**: Lucide Font (иконки)
- **API**: MUIP (взаимодействие с сервером через подписанные GET-запросы)
- **Нейросети**: проект создан с помощью нейросети

##  Быстрый старт (для тех, кто не ищет легких путей)

### 1. Клонирование репозитория

```
git clone https://github.com/azfdagg/muip-gui.git
cd muip-gui
```

### 2. Установка зависимостей

```
pip install -r requirements.txt
```


### 3. Настройка конфигурации

Создайте файл `data/config.json`:
(по умолчанию он уже существует и имеет стандартные настройки)

```
{
  "muip": {
    "dev_docker": {
      "host": "http://127.0.0.1:21051",
      "sign": "ваш_секретный_ключ"
    }
  }
}
```

### 4. Запуск

```
python app.py
```

Откройте в браузере: [http://127.0.0.1:8483](http://127.0.0.1:8483)
*Адрес можно поменять на 0.0.0.0 в конце python файла. Тогда вы сможете пользоваться интерфейсом с любого устройства.*

## 🔧 MUIP API

Панель поддерживает **все MUIP обработчики** (1005, 1116, 1127, и др.).
Каждый запрос автоматически подписывается:

```
1. Сортировка параметров по алфавиту
2. Формирование строки query: key1=value1&key2=value2...
3. SHA256(query_string + secret_key)
4. Добавление подписи в URL: &sign=...
```

## 🧠 Справочник (Handbook)

- Автоматически парсит все `.txt` файлы в формате `ID : Название`
- Поддерживает поиск по ID и названию
- Можно выдавать предметы напрямую из таблицы (1 шт или 100 шт)
- Для персонажей автоматически используется команда `avatar add`

## 📦 Примеры команд

```
# Выдача предмета
item add 201 10000

# Выдача персонажа
avatar add 10000002

# Выдача оружия (уровень 90, пробуждение 5)
equip add 13501 90 5

# Выдача артефакта (кастомные статы)
/give 76544 lv21 x1 15001 501034,2 501054,1

# Телепорт по координатам
goto 2748 -1450 200

# Спавн монстра
monster 20010101 3 90

# Отправка почты (через MUIP 1005)
# Реализовано через интерфейс
```

## 📝 Примечания

- Для работы требуется **запущенный сервер** с MUIP API 
- Артефакты на **старых версиях** сервера могут не поддерживать кастомные уровни/статы
- Иконки используются из **Lucide**
- Все ответы сервера логируются в **консоль**
